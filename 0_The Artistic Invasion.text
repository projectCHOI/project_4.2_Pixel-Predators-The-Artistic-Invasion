# 추가내용
# 이미지를 넣자.
1. PNG : 게임의 배경 및 기타 요소에도 사용하기 좋다.
- 가장 일반적으로 사용되는 이미지 형식
- 투명도 지원
- 손실 없는 압축 제공 = 이미지 품질이 유지

2. JPG : 플레이어나 주요 게임 요소에는 부적합
- 손실 압축을 사용하는 형식 = 파일 크기가 상대적으로 작다
- 배경 이미지에 적합
- 투명도 미지원

3. 게임의 스테이지 구성
- 다른 자연환경 배경의 스테이지 13개
- 날아오는 적을 피하거나 제거
- 보스가 나타나면 보스 스테이지 시작
- 보스 처치로 보석 휙득 후 다음 스테이지 이동

4. 점수
- 적을 처치시 1점
- 모든 스테이지 클리어 조건 12개의 보석 수집
- 스코어 적 처치 점수
